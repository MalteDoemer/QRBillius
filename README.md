# QRBillius

This is a GUI application written in java to create Swiss QR-Code bills. This project is based on the [SwissQRBill](https://github.com/manuelbl/SwissQRBill) library.

## Features

- Create single QR bills manually
- Create multiple bills based on information from a csv or xlsx file
- Save the bills as PDF
- Save bills using a docx template (mail merge)

## Download

Installers for windows are available at: https://github.com/MalteDoemer/QRBillius/releases
